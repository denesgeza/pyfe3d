Finite elements for 3D problems in Python/Cython¶

  • Beam property module (pyfe3d.beamprop)
  • Shell property module (pyfe3d.shellprop)
  • BeamC - Consistent Timoshenko 3D beam element (:mod: \(pyfe3d.beamc\))
  • BeamLR - Linear Timoshenko 3D beam element with reduced integration (:mod: \(pyfe3d.beamlr\))
  • Truss - Linear truss 3D element with analytical integration (pyfe3d.truss)
  • Quad4R - Quadrilateral element with reduced integration (pyfe3d.quad4r)
  • Quad4R useful functions (pyfe3d.quad4r_utils)
  • Coordinate systems (pyfe3d.coord)
